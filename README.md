# SwitchingBetweenMonitorsScriptAutoHotKeys
A script to switch between monitors using AutoHotkeys v2


This script moves your mouse automatically to whichever monitor you want to focus on in windows. 

Untested so use at your own discretion. 

Ctrl + Alt + 1 → focus monitor 1

Ctrl + Alt + 2 → focus monitor 2

You can modify the script for how many monitors you have.
